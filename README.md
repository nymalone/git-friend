# Git Friend

An simple platform to you search for yours friends and check their repos.

## Getting Started
* Clone or dowload this repo
* Run `npm install`
* Run `npm start`

## Built With
To make this site we use:

* JS ES6
* Node.js
* React.js
* HTML5
* Styled Components
